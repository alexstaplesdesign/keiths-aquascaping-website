# Keith's Aquascaping Services

Website for a local aquascaping business. Service pages, a portfolio
gallery, and a contact form.

## Stack

HTML5 · CSS3 · JavaScript (vanilla) · Gabriel Sans (custom font)

## Running it

Open `index.html` in a browser. No build step.

Live demo: [view site](https://alexstaplesdesign.github.io/keiths-aquascaping-website/)

## Pages

- `index.html` — Homepage
- `about.html` — About Keith
- `services.html` — Service offerings
- `contact.html` — Contact form
